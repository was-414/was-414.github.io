Markdown学习记录

2021年01月12日

Markdown是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。Markdown编写的文档的后缀名为.md。下面我将记录Markdown的常用语法。

标题

用#来表示1~6级标题，几级标题对应有几个#号，例如:

# 一级标题
### 三级标题

注意事项:#号后面要加一个空格。

段落格式

换行
两个空格加一个回车。
分割线
三个及三个以上的减号（-）号。
删除线

~~删除的内容~~

下划线

<u>加下划线的内容</u>

脚注

内容[^test]
[^text]:内容的意思

列表

无序列表
用星号(*)、加号(+)或是减号(-)作为列表标记，后面加一个空格。
有序列表
数字1加一个点（.）。
列表嵌套
子列表前加4个空格。

区块

>加一个空格。

> 区块引用
> 今天天气真好！

代码块

`代码`
```
代码段
```

超链接

用法如下:

用法一:
[链接名称](链接地址)  或  <链接地址>
用法二:
通过变量来设置一个链接，变量赋值在文档末尾进行
这个链接用 1 作为网址变量 [Google][1]
这个链接用 runoob 作为网址变量 [Runoob][runoob]
然后在文档的结尾为变量赋值（网址）
  [1]: http://www.google.com/
  [runoob]: http://www.runoob.com/

图片

![alt 属性文本](图片地址)

![alt 属性文本](图片地址 "可选标题")

表格

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

说明:可以设置表格的对齐方式

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

:-表示左对齐，:-:表示居中对齐,:-表示右对齐。
