linux基本命令

命令格式：命令 -选项 参数 （选项和参数可以为空）如：ls -la /usr

命令 参数 示例 说明
reboot reboot 重启
init 0 init 0 关机
shutdown -h now shutdown -h now 关机
ifconfig ifconfig 查看ip地址
ping ping www.baidu.com 测试网络联通性
passwd 空（改自己） passwd 更改自己密码
其他用户 passwd centos 更改其他用户密码（管理员操作）
date date 显示系统日期
cal 空 cal 显示当前日历
月，年 cal 10 2008 显示2008年10月的日历
who who 列出当前系统所有用户
who am i who am i 显示你的登录用户名
clear clear 清屏
echo 字符串 echo hello world 在屏幕上输出hello world
mesg y mesg y 确保自己能收到消息
write 用户 write root 向root发消息,写完ctrl+d结束消息编辑
history history 查看过去的历史命令
数字 history 20 查看过去的20条历史命令
! 数字 !5 执行历史命令列表的第5条命令
exit exit 退出登录到终端的用户
logout logout 退出登录到终端的用户

文件管理相关命令

命令 参数 示例 说明
cat cat 1.txt 查看文件内容
cat 1.txt 2.txt 合并文件并查看内容
-n cat -n 1.txt 显示行号
-A or -E cat -A 1.txt -A显示所有不可见字符，-E显示行结束符
more more 1.txt 分页显示，按enter向后移动一行，按空格向后移动一页,按b向前移动到上一屏，按q退出
-num more -10 1.txt 表示一个屏幕显示该数字的行数
-p more -p 1.txt 显示下一屏之前先清屏
+num more +4 1.txt 从该数字行开始显示
-d more -d 1.txt 底部显示友好信息
less less 1.txt 分页显示，和more类似(参数相同写法)，有查找功能
less 1.txt 查询关键字时，先输入/，在输入关键字
head head 1.txt 显示文件开头若干行
-num head -10 1.txt 显示文件开头10行
tail tail 1.txt 显示文件末尾若干行
-num tail -10 1.txt 显示文件末尾10行
touch touch str.txt 创建空文件（该文件不存在）
-t touch -t 201809011230.10 1.txt 修改文件日期
grep grep root 1.txt 查找文件或目录中含指定字符串的行
-i grep -i root 1.txt 不区分大小写
-n grep -n root 1.txt 输出时加上对应的行号
-c grep -c root 1.txt 输出与字符串相匹配的行的数目
-v grep -v root 1.txt 输出不相匹配的行
-x grep -x root 1.txt 严格匹配
-r grep -r root dir 递归查询（查目录）
sort sort 1.txt 对文本的各行排序
-r sort -r 1.txt 逆序排
-o sort 1.txt -o 2.txt 将结果输出到指定文件中
-n sort -n 1.txt 按数值大小来排序
-k sort -k 1.txt 指定每一行中第几个字符为排序关键字
uniq uniq 1.txt 比较相邻行，去掉重复的行
-c uniq -c 1.txt 输出时，在行首显示该行在文本里出现的次数
-d uniq -d 1.txt 只显示重复的行
-u uniq -u 1.txt 只显示不重复的行
cp cp 1.txt 2.txt 将源文件或目录复制到指定文件或目录(指定文件不存在会自动创建)
-p cp -p 1.txt 2.txt 除复制文件外，还将存储权限和修改时间一并复制
-r or -R cp -r dir1 dir2 递归复制（针对目录）
-a cp -a dir1 dir2 相当于 cp -rp dir1 dir2
rm rm 1.txt 删除文件或目录
-r or -R rm -r dir 递归删
-f rm -f 1.txt 强制删除，不给出提示信息
-i rm -i 1.txt 删除前先询问用户
-v rm -v 1.txt 显示指令执行过程
mv mv 1.txt .. 移动或更名指定的文件或目录
-b mv -b 1.txt .. 覆盖前会备份原有文件
-f mv -bf 1.txt .. 强制覆盖，不给提示信息
-i mv -i 1.txt .. 覆盖前先询问用户
-v mv -v 1.txt .. 显示指令执行过程
wc wc 1.txt 2.txt 统计文件内容
-c -l -w wc -clw 1.txt 字节数、行数、字数
mkdir mkdir ./t1 创建目录
-p mkdir ./t1/t2 创建多级目录
-m mkdir -m 755 ./t1 创建目录的同时赋予权限
rmdir rm t1 删除目录
-p rm t1/t2/t3 删除多级目录（没有rm好用）
cd cd .. or cd dir1 切换目录
pwd pwd 显示当前目录
ls ls 列出当前目录的内容，也可以自己指定路径
-l ls -l or ll 以长格式（详细格式）显示
-a or -A ls -a 显示所有文件和目录，-A不会显示.(当前目录)和..(上级目录)
-R ls -R 递归处理
ln ln 1.txt 1t 建立链接，链接分为硬链接（相当于别名）和软链接（相当于快捷方式）
-s ln -s 1.txt 1t 加上-s是软链接方式，不加是硬链接方式

其他常用命令

文件权限

chmod    改变文件或目录的权限
语法：chmod [who] [+-=] [mode] 文件名
[who]可以为一下的内容中的一个或是组合
u 文件主，创建这个文件的主人
g 组
o 其他用户
a 所有用户
[+-=]
+ 增加权限
- 减去权限
= 赋值权限
[mode]权限，分为下面三种
r 可读
w 可写
x 可执行
列子：
chmod u+x ex1
chmod u+rwx,o-rw ex1
chmod u=rwx,g=rx,o=r ex1

另外的一种赋权限的方式,用二进制方式
r权限为 4
w权限为 2
x权限为 1
chmod 754 ex1     等价于     chmod u=rwx,g=rx,o=r ex1
命令 参数 示例 说明
chown chown centos:centos 1.txt 改变文件的所有者或文件的组
-R chown -R root:root dir 递归改变指定目录和其子目录
chgrp chgrp centos 1.txt 改变文件所属的组
-R chgrp -R root dir 递归改变指定目录和其子目录

命令帮助

命令 参数 示例 说明
man man ls 查阅指定命令的手册
-f man -f ls 查看该命令有那些章节
-S man -S 1 ls 查看命令指定章节的帮助
help help cd shell内置命令帮助
ls –help 另外一种关于help的用法

进程管理

命令 参数 示例 说明
ps ps 报告程序的状况
a u x ps aux a 显示所有终端机下执行的程序，u 面向用户的格式，x 显示所有非控制终端的进程信息
-e -f ps -ef -e 此参数的效果和指定”A”参数相同，-f 显示进程的所有信息
kill kill （pid） 删除执行中的程序或工作
-1 -9 kill -9 (pid) -1，查看信号；-9，强制删除
top top 动态的查看进程状态
pstree pstree 以树形的方式查看进程

压缩解压缩

gizp     .gz 的压缩方式
-num   调整解压速度
    -1 or --fast 表示最快的压缩方法（低压缩比）
    -9 or --best 表示最慢的压缩方法（高压缩比）
-d 解压缩
-r 递归
zcat  不解压查看.gz文件的内容

bzip2    .bz2 的压缩方式,无递归的方式
-k 不删除源文件
-d 解压缩
bzcat  不解压查看.bz2文件的内容

xz       .xz 的压缩方式,无递归的方式
-d 解压缩
-k 不删除源文件

tar 打包
-c 创建包
-t 查看包中的文件
-x 释放包
-z .gz格式压缩包
-j .bz2格式压缩包
-J .xz格式压缩包
-f 指定包名（带一个参数）
-v 执行过程中显示提示信息
-C 用于指定包解压缩释放的路径
列子：(以哪种方式来压缩包的参数放第一个,剩下的的参数为三种cvf(创建)、tvf(查看)、xvf(释放))
tar -zcvf t1.tar.gz 1.txt 2.txt 3.txt    把1.txt,2.txt,3.txt打包成t1.tar.gz并以.gz格式来压缩
tar -tvf t1.tar.gz      查看t1.tar.gz 里的文件名
tar -zxvf t1.tar.gz     以.gz格式来释放包t1.tar.gz
